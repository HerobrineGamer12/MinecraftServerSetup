โปรแกรมนี้จัดทำขึ้นเพื่อเปิดโอกาสให้ผู้ที่ไม่สามารถเปิดเซิฟเวอร์
Minecraft ได้ 
ทำให้สามารถทำได้ง่ายขึ้น .


======== English [version 1.7.2 up] ========
1.open MinecraftServerSetup.exe
2.select version
3.Click Download Server Button
4.wait Download = 100% 
5.select max-ram-server
6.Click Create Button
7.edit config or auto config
8.start server
9.wait 15 s.
10.close terminal
11.click enable lula
12.start server button !
13.Enjoy ! XD
======== ภาษาไทย ========
วิธีใช้งาน
 [version 1.7.2 ขึ้นไป]
1.เปิดโปรแกรม MinecraftServerSetup

2.เลือก Version ที่ต้องการ

3.กดปุ่ม Download Server

4.หลังจาก แถบ Download เต็ม 100% เลือก Max Ram Server
ที่ต้องการ มาตรฐานคือ 1024

5.กดปุ่ม Create

6.แก้ไข Config ที่หัวข้อที่ 3 หากตั้งค่าไม่เป็นให้เลือก auto config
7.กดปุ่ม Start Server รอจนขึ้น PAUSE
8.หลังจากนั้นกดปุ่ม Enable Lula 
9.จากนั้น Start Server อีกครั้ง 
10. Enjoy ! XD
